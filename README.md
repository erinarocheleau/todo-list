# Todo Web App

A simple, modern Todo app that runs entirely in the browser.  
Your tasks are **automatically saved to local storage**, so they persist after refresh.

## Features

- **Add / delete tasks**
- **Toggle completed / active**
- **Filtering** (All / Active / Completed)
- **Clear completed tasks**
- **Light / dark theme toggle** (theme is also persisted)

## Usage

1. Open this folder (`todo`) in VS Code / Cursor (or any editor).
2. You can open `index.html` directly in your browser and it will work.
3. Or, if you have Node.js installed, you can run a tiny local server:

```bash
cd todo
npm install serve -D
npm run start
```

Then open the URL shown in the terminal (for example `http://localhost:3000`).
